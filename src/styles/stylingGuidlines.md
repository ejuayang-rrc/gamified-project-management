# Styling Guide

Include any global styling rules to follow when styling common components.

- font
- font size,
- color palette,
- etc.

## tailwind.css

This file is where tailwind is initialized and is applied across the **whole project**.

**DO NOT** import tailwind in any other file.

## components.css

feel free to make base styles for components if your inline styles get messy.

make this file later if needed.

[Refer to these docs to see how.](https://tailwindcss.com/docs/adding-custom-styles#adding-base-styles)
