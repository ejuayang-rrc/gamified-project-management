# Style Guide

The following style guide covers:

- the color scheme or pallet
- Typography
- Layout Style and Design (TBA  as a section in the future.)
- Coding Style

## Color Scheme

- D8382F (Red)
- 545454 (Dark Gray)
- D6D8D7 (Light Gray)
- FFFFFF (White)

These colors build up the background. Tasks colors will be determined later.

## Typography

- Headers: Latino Gothic Variable
- Paragraph font: Faustina

## Coding Style

- Single line break where applicable
- <https://ts.dev/style/#identifiers>
- Commit message naming convention:
  - “Feat: {feature-message}”
  - “Fix: {fix-message}”
- Pull Request naming convention:
  - “Feat: {main-feature-message}”
  - “Fix: {main-fix-message}”
